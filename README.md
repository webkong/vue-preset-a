# vue-cli-multiProject

> 多项目分别打包，支持环境变量配置

```
"axios": "^0.19.0",
"core-js": "^3.4.0",
"regenerator-runtime": "^0.13.3",
"vue": "^2.6.10",
"vue-i18n": "^8.15.0"
```

### Usage

This is a project template for [vue-cli](https://cli.vuejs.org/zh/).

``` bash
npm install -g @vue/cli
vue create webkong/vue-preset-a my-project
npm run serve
```
